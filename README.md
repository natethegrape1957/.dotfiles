# Dotfiles Info
## Requirements:
- Starship:
```bash
curl -sS https://starship.rs/install.sh | sh

# ~/.bashrc
eval "$(starship init bash)"
```
- Install Homebrew:
```bash
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
```
- Install nvim
```bash
brew install neovim
```

```bash
sudo snap install nvim
```
